STEP 1 - Install Node.js
  • Download dari: https://nodejs.org/
  • Pilih "LTS" → Install → Restart komputer

STEP 2 - Extract & Setup Project
  • Extract semua file ke folder, misalnya: C:\soal-bimasoft\
  • Buka Command Prompt (Windows) atau Terminal (Mac/Linux)
  • Ketik: cd C:\soal-bimasoft

STEP 3 - Install Dependencies
  • Ketik: npm install
  • Tunggu sampai selesai (~2-3 menit)

STEP 4 - Jalankan Aplikasi
  • Ketik: npm run dev
  • Browser otomatis membuka: http://localhost:5173
  • Selesai! Aplikasi siap digunakan 🚀
