Panduan Pembaruan Aplikasi

Generator Template Soal Bimasoft v.1.3.0

Dokumen ini menjelaskan langkah-langkah bagi pengembang untuk merilis versi terbaru (update) aplikasi sehingga pengguna yang sudah menginstal versi sebelumnya otomatis menerima pembaruan secara instan.

Langkah 1

Naikkan Versi di Kode Sumber

  1. Buka berkas package.json di folder utama proyek Anda.
  2. Cari properti "version" (biasanya di baris ke-3) dan ubah nilainya ke versi baru yang lebih tinggi (misalnya dari "1.0.0" menjadi "1.1.0").
"version": "1.1.0"
  • Tips: Jika Anda ingin memperbarui catatan perubahan atau judul versi pada jendela dialog Tentang Aplikasi, silakan sesuaikan teks versinya di berkas src/App.jsx.
Langkah 2

Bangun (Build) Paket Desktop Baru

Jalankan perintah kompilasi berikut pada terminal/command prompt proyek Anda:

npm run electron:build

Perintah ini akan secara otomatis mengompilasi ulang kode sumber dan menghasilkan dua berkas penting di folder dist-desktop/:

📦 Generator Template Soal Bimasoft Setup 1.1.0.exe (Installer utama baru)
📄 latest.yml (Metadata verifikasi pembaruan)
Langkah 3

Unggah Pembaruan ke GitHub Releases

  1. Buka halaman GitHub repositori Anda di browser:
    https://github.com/fadlirahman1987/template-soal-bimasoft
  2. Di sisi kanan halaman repositori, klik menu Releases.
  3. Klik tombol "Draft a new release".
  4. Pada bagian "Choose a tag", ketik tag versi baru Anda (contoh: v1.1.0) dan klik Create new tag.
  5. Tulis judul pembaruan (contoh: Rilis Pembaruan v1.1.0).
  6. Pada kolom unggah berkas rilis, seret (drag & drop) dua berkas berikut dari folder lokal /dist-desktop/:
    • Generator Template Soal Bimasoft Setup 1.1.0.exe
    • latest.yml (Wajib diunggah agar auto-update berjalan!)
  7. Klik tombol "Publish release" untuk memublikasikan pembaruan Anda.
Langkah 4

Cara Kerja Pembaruan di Pengguna

  1. Saat pengguna membuka aplikasi versi lama (misal 1.0.0), aplikasi akan mendeteksi info rilis baru pada berkas latest.yml di repositori GitHub Anda.
  2. Aplikasi mengunduh installer versi baru secara otomatis dan senyap di latar belakang.
  3. Notifikasi sistem operasi Windows akan muncul saat unduhan selesai.
  4. Saat pengguna menutup dan membuka kembali aplikasi, versi baru (1.1.0) sudah langsung terpasang otomatis.
